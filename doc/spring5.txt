WEB
웹은 아래와 같은 형식으로 발전해 왔다.
HTML - Servlet - JSP - MVC - MVC FrameWork

HTML            순수 HTML만으로 작성되었다.
Servlet         자바기술로 백엔드단에 HTML을 작성하였다.
JSP             HTML문서에 java코드를 작성하였다.
MVC             프론트와 백을 구분하여 HTML과 java를 분리 작성하였다.(계층구분 시작)
MVC FrameWork   MVC패턴을 기준으로 다양한 기능을 표준으로 넣은 프레임 워크로 개발을 시작한다.

예전엔 파일로 데이터를 저장했으나, 네트워크가 발전하면서 DB의 활용이 많아졌다.
